# Tailor Shop Management Backend

This is a FastAPI backend for managing a tailor shop, supporting multiple shops, customers, measurements, pieces, stitchers, assignments, and users.

## Features

- Shop management (Coats, Pants, Shalwar Kameez, Sherwani, etc.)
- Customer management
- Measurements tracking
- Piece tracking
- Stitcher assignments
- User management with role-based access (Owner, Staff)
- JWT authentication
